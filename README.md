# Knowledge Library

> [!NOTE]
> これはコンセプト検証用のサンプルリポジトリです。実際の運用に向けた叩き台として参照してください。

SlackやViva Engageで共有された書籍・動画・論文がフロー情報として流れてしまう課題を解決するための知識蓄積リポジトリ。  
紹介されたコンテンツをストック情報として残し、チームの知的資産として活用することを目的としています。

## コントリビュート

新しいコンテンツを追加する場合は以下の手順で：

1. `library/` 配下に `{種別}_{タイトル}.md` を作成（テンプレートは既存ファイルを参照）
2. このREADMEの該当セクションにリンクを追加
3. PRを作成してレビューを依頼
