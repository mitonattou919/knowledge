## 概要

Claude Code（AnthropicのAIコーディングCLIツール）が持つ公式スキル・プラグイン機能を網羅的に紹介した動画。

Claude Codeはコード生成だけでなく、スキルと呼ばれる拡張機能を通じてワークフロー自動化・外部サービス連携・繰り返しタスクの自動化などを実現できる。どんなスキルがあるか・どう使うかを一通り把握できる内容で、Claude Code導入初期のキャッチアップに最適。

## メモ・考察

「Claude Codeって何ができるの？」という問いに答えるための動画として、開発チームへの紹介やオンボーディング資料として活用できる。
